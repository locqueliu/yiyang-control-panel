# Security Policy

## 支持范围

当前仓库仍处于首版工程阶段，默认只对 `main` 分支上的最新代码进行问题确认，不承诺对历史快照提供补丁支持。

## 报告安全问题

如果你发现以下问题，请不要直接公开提交 Issue：

- 未授权访问设备控制接口
- 敏感信息泄露
- 可被利用的局域网扫描或命令执行问题
- 配置导入导出导致的注入风险

建议采用以下方式处理：

1. 先整理复现步骤、影响范围、固件版本和日志
2. 不要附带真实设备的公网信息、局域网地址、密钥或账号信息
3. 在私下沟通渠道建立后，再提供完整细节

## 临时处理建议

- 在问题修复前，不要将调试构建长期安装在不受信任网络环境中
- 不要把包含真实设备数据的导出配置直接公开上传
- 对高风险接口的修改应在隔离环境下验证
