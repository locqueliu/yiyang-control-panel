# Code of Conduct

## 我们希望的协作方式

这个项目欢迎认真、直接、愿意解决问题的人参与。

我们鼓励：

- 针对问题本身讨论，而不是针对个人
- 提供可复现信息、上下文和建设性建议
- 尊重不同经验层级的贡献者
- 在指出问题时保持克制和清晰

我们不接受：

- 人身攻击、侮辱、羞辱或持续性挑衅
- 恶意刷屏、故意制造无效讨论
- 未经允许公开他人的隐私、凭据或设备信息
- 明知会破坏项目协作仍反复进行的行为

## 执行方式

维护者有权对不符合上述原则的内容进行编辑、隐藏、关闭或移除，并视情况限制后续参与。

如果你在协作中感到被冒犯或发现明显越界行为，请通过合适渠道向维护者说明情况，并尽量提供上下文。
